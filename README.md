### Vite / Bootstrap5 template with pnpm


#### install

```shell
pnpm instlal
```

#### build

```shell
pnpm build
```

#### preview

```shell
pnpm preview
```

#### Structure

```
.
├── README.md
├── package.json
├── pnpm-lock.yaml
├── public
├── src
│   ├── index.html
│   ├── js
│   │   └── main.js
│   └── scss

```
